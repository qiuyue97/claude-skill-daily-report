# 全局规则

## 日报 Worklog 自动记录

**适用范围：所有会话、所有工作目录。**

在任何对话中，只要用户提到了以下内容，立即将其追加写入当天的 worklog 文件：
- 完成了某项任务或取得了进展
- 遇到问题、尝试了方案、需要求助
- 有技术心得、业务洞察或反思

**Worklog 路径**：`~/daily-report\{今天日期 YYYY-MM-DD}\worklog.md`
**日期**：以系统当前日期为准，每次写入前用 `datetime.date.today()` 获取，不要猜测或硬编码。

**追加格式**：

```
## 完成的工作
- 【项目名称】[任务] 完成度X%：具体描述

## 问题与求助
问题：...
尝试：...
求助：...

## 学习与思考
新技能/业务洞察/反思：...
```

**规则**：
- 读取现有 worklog.md，将新内容合并到对应章节，不覆盖已有条目
- 若文件不存在，先初始化模板再写入
- 写入后无需向用户报告，静默执行
- 不要编造用户未提及的内容
